# SIGHIVE

Special Interest Group of HIVE (`SIGHIVE`)是HIVE实验室的特别兴趣小组，旨在通过互相分享快速学习更多图形学的前沿知识。分享内容包括但不限于:

- Paper研读总结
  - Global Illumination
  - Ray Tracing
  - Real-Time Rendering
  - Physical-based Rendering
  - Others
- 《Real-time rendering》
- 《PBRT》
- 有趣的技术
- 独到的经验
